About

The Language Learning website is a language learning tool designed to assist a language learner generate learning content that suits their needs and interests. The goal of this site is to give the language learner controller over the content that they deem relevant and make translating content enjoyable and convenient. The site does not contain structured lessons but rather shared and private content that the users create.

The desigers of the backend are Craig E Carson, Joshua Ray Chapman, and Jeffrey Scott Patton. The front end designer was Corey Teeter as well as Jeffrey Scott Patton

This website was developed using the following

  • Python and Django
  • HTML CSS and JavaScript
  • SQL and SQLlite
  • Json, PyCharm, Pytesseract,
  • PyDictionary, Oxford Dictonary, Google Translate
